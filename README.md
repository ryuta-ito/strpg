# About
strpg (strict programming) is the tool.

# Instration
$ git clone <git-url>
$ cd strpg
$ gem install strpg-0.0.0.gem

# Usage

## open development environment in accordance with strpg.yml
`strpg developmenting`

## show task
`strpg task`

## unwritable and make files
`strpg strict`

## writable all files
`strpg strict --reset`

## show task
`strpg task`
