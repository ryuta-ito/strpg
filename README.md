# About
strpg (strict programming) is a tool to support your programming.

# Installation
```
$ git clone 
$ cd strpg
$ ./bin/release
$ gem install strpg-0.0.0.gem
```

# Usage

## open development environment in accordance with strpg.yml
`strpg developmenting`

## show task
`strpg task`

## unwritable and make files
`strpg strict`

## writable all files
`strpg strict --reset`
